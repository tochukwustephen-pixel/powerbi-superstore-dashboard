# Power BI Superstore Sales Dashboard

A professional 5-page interactive dashboard built in Power BI Desktop.

## Dataset
Sample Superstore — 9,994 rows, 21 columns

## Dashboard Pages
1. Executive Summary — KPI cards, sales by category, profit by region
2. Sales & Profit Trends — time series analysis, order growth
3. Regional & Segment Analysis — filled map, segment breakdown
4. Product Performance — sub-category profitability, discount analysis
5. Customer & Shipping Analysis — top 10 customers, shipping methods

## Key Insights
- Total Revenue: $2.3M | Total Profit: $286K | Profit Margin: 12.47%
- Tables sub-category lost $17,725 across all orders
- Discounts above 20% produce negative average profit
- West region leads with 14.9% profit margin vs Central at 7.9%

## Tools Used
- Power BI Desktop
- DAX (8 measures including time intelligence)
- Power Query for data transformation
